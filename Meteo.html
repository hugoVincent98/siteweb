Application métérologique

Mes projets

Le projet consistait à créer une application web qui permettra aux utilisateurs de visualiser de différentes manières les données météo historiques de différentes villes canadiennes.L'objectif de l'application était de permettre aux utilisateurs de consulter rapidement l'ensemble des villes (menu de gauche), classées par province. Après avoir sélectionné une ville, l'usager peut ensuite choisir une période cible, puis afficher les données météorologiques correspondant à la période choisie. L'application affichera également certaines statistiques de base.


Certaines des fonctionnalités qui ont été implémentées sont :

  • La liste des stations météo, classées par province
  • Les données météorologiques pour chaque station, par mois/année
  • Des statistiques météo globales et par mois
  • Des informations météo historiques pour une journée donnée ("voyage dans le temps")
  • Les prévisions météo pour aujourd'hui et les prochains jours
  • Implémentation d'une API REST
  • Implémentation d'une cache structuré des requêtes à l'API
  • Implémentation d'une affichage des données météo sur une carte


    • Dans ce projet, nous avons utilisé:

  • Vue.js
  • NodeJs
  • Axios
  • Leaflet
  • MongoDB
  • AJAX
  • XML
  • JSON
